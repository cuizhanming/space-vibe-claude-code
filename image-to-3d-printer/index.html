How it works

  1. Upload an image (JPG, PNG, etc.)
  2. The app converts the image to a grayscale height map
  3. Adjust settings to customize your 3D model
  4. Preview the 3D model in real-time
  5. Download the STL file for 3D printing

Tip: Images with high contrast work best. Lighter areas become raised surfaces, darker areas become recessed.
